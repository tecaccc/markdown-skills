```
Top:  标题背景.jpeg + 标题背景文字.png (same as Page 2)

```

**Title bar:** Same as Page 2.

**Subtitle:**
- Position: x=46px, y=180px

**Content area:**

- Region: x=46, y=220, width=1188, height=440 (可用排版区域)
- AI 应根据当前页的实际内容，自主选择最合适的排版方式，核心原则：**信息量充分 + 视觉美观**

**可选排版方式（不限于以下，可灵活组合）：**

| 排版 | 适用场景 | 说明 |
|------|----------|------|
| 单栏全宽 | 长段落、完整论述、流程说明 | 文字占满内容区宽度，适合阅读性强的内容 |
| 双栏并列 | 对比、分类、两组并行信息 | 左右各约 50% 宽度，间距 30-40px |
| 左图右文 / 左文右图 | 图文搭配、案例展示 | 图片约 40-50% 宽度，文字占另一侧 |
| 三栏等分 | 多个并列要点、特性列表 | 每栏约 33% 宽度，适合 3 个关键词/图标 |
| 上下分区 | 标题+详细内容、总分结构 | 上方放概要/关键数据，下方放详细说明 |
| 大图全屏 | 架构图、流程图、数据可视化 | 图片占满内容区，仅保留必要标注文字 |

**排版决策原则：**
1. **内容优先**：排版服务于内容表达，不要为了形式而牺牲信息完整性
2. **视觉平衡**：页面不要过于拥挤或过于空旷，合理利用留白
3. **一致性**：同一章节内的内容页风格尽量统一
4. **层次清晰**：通过字号、加粗、颜色区分标题、正文、注释等层级

**通用样式：**
- 正文：微软雅黑 16pt, Color: #08287F
- 小标题/强调：微软雅黑 18-20pt, Bold, Color: #08287F
- 注释/辅助文字：微软雅黑 12-14pt, Color: #4A6FA5
- 图片：保持宽高比，不拉伸变形
- 内容区域无背景填充，保持透明
